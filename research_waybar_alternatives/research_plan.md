# Research Plan: Waybar Alternatives

## Main Question

What are good modern replacements for Waybar, especially for a Niri-based Linux desktop, and what are the practical trade-offs for this repository?

## Subtopics

### 1. Current status of Wayland bar alternatives
- Identify active alternatives to Waybar for Linux/Wayland desktops.
- Check project activity, positioning, and configuration model.
- Focus on bars suitable for daily desktop use rather than demos.

### 2. Niri compatibility and integration
- Check whether promising alternatives work well with Niri.
- Look for support for workspaces, window info, tray/status modules, notifications, and custom commands.
- Note gaps compared with Waybar.

## Expected Output
- A short ranked list of practical replacements.
- Guidance on which options are more modern versus just more niche.
- A recommendation tailored to this repo's current static profile system.

## Synthesis Approach
- Compare alternatives on maturity, Wayland support, Niri friendliness, theming flexibility, and migration effort.
- Prefer options that are realistic to adopt in this codebase without rebuilding the whole desktop workflow.
