## Try it out yourself!

### 1. Download the ISO from https://nix.rupan.dev

### 2. Flash the ISO onto a USB flash drive

### 3. Boot into the flash drive

### 4. Connect to wifi:
```
nmtui
```

### 5. Run the get-config command:
```
get-config
```

### Done!
